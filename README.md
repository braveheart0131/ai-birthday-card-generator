# AI Birthday Card Generator (OpenAI + Vercel + GitHub Pages)

AI Birthday Card Generator

An AI-powered web app that generates personalized birthday messages.

Tech Stack
- HTML / CSS / JavaScript
- Serverless backend using Vercel
- OpenAI API for message generation
- GitHub Pages for frontend hosting

Features
- AI-generated birthday messages
- Humor style selection
- Optional hobbies
- Copy / WhatsApp / SMS / Email sharing
- Confetti animation

Architecture
Frontend → GitHub Pages
Backend → Vercel Serverless API
AI → OpenAI
